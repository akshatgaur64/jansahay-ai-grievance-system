# JanSahay – AI Governance Intelligence System

JanSahay is an AI-driven grievance redressal and welfare eligibility platform designed to reduce administrative latency using hybrid machine intelligence.

It integrates Natural Language Processing and Predictive Analytics to automate grievance routing and recommend welfare schemes efficiently.

## 🧠 System Architecture

### Kernel A – NLP Grievance Engine
• TF-IDF vectorization  
• Multinomial Naive Bayes classification  
• Heuristic priority scoring system  

### Kernel B – Welfare Recommendation Engine
• Random Forest classifier  
• Demographic-based prediction  
• Confidence-based ranking  

### Hybrid Logic Layer
• Deterministic routing override  
• Non-blocking workflow design
